Tintap: The game of tints and shades. (Game 2018)

     

    Tintap is the game about the difference between the colors that’s look like same. As we all know the human eye can distinguish about 10 million different colors. So know what about you. Ever you imagine the possibility with colors that your eye can see?

    This game is design with this simple concept in mind that human eye can see the colors differently , in this game you just need to select on color from given four different color , this color must be same as the color of drop that fall from top.

    If the selected color is same as the color of drop then you will get the point else the game will be over. So don’t try just play, but try to learn something new and you will get the knowledge about our eye can see the different between colors.

    Think different, play different & see different.

    So if you try you will learn, if you don’t you will not see the possibility!
